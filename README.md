# Simple-guess-game

**How to setup**

## Folow the 3 steps

- Download the source code:
-     git clone https://github.com/thanasispe/simple-guess-game.git
      

- cd into the source directory:
-     cd simple-guess-game/src

- Compile the source code:
-     make set all

- Run the game:
-     cd ../bin && ./guess-game
